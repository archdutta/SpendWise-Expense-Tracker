#  SpendWise

Track Smarter. Save Better.

A FastAPI-powered budgeting and expense tracking platform that helps users manage expenses, set monthly budgets, analyze spending patterns, and generate financial reports.

## Features

* JWT Authentication
* User Signup & Login
* Expense Management (CRUD)
* Monthly Budget Tracking
* Dashboard Analytics
* Spending by Category Pie Chart
* Monthly Overview Bar Chart
* Financial Reports
* SQLite Database

## Tech Stack

* FastAPI
* Python
* SQLAlchemy
* SQLite
* Jinja2
* Bootstrap 5
* Chart.js
* JWT Authentication
